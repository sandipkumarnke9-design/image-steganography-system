# 🔐 Advanced Image Steganography System  

> 🖼️ A web-based application that securely hides and extracts confidential data within images using steganography techniques and optional password encryption.

---

## 🚀 About the Project  

The **Advanced Image Steganography System** is an interactive web application that enables users to embed and extract hidden messages inside digital images.

It uses modern steganography techniques to ensure secure and invisible data transmission.

This project demonstrates:
- Data security concepts  
- Image processing using JavaScript  
- Encryption techniques  
- Interactive UI/UX design  

---

## ✨ Key Features  

- 🔒 **Encode Mode** – Hide secret messages inside images  
- 🔓 **Decode Mode** – Extract hidden messages  
- 🔐 **Password Protection (Encryption)**  
- 🧠 **Multiple Techniques**
  - LSB (Least Significant Bit)  
  - DCT (Discrete Cosine Transform)  
- 📤 **Image Upload & Preview**  
- 💾 **Download Processed Image**  
- ⚡ **Real-time Processing (Client-side)**  

---

## 🛠️ Tech Stack  

- **Frontend:** HTML, CSS, JavaScript  
- **Image Processing:** Canvas API  
- **Encryption:** XOR-based + Base64  
- **Environment:** Browser (No backend required)  

---

## 🧠 How It Works  

### 🔹 Encoding Process:
1. User uploads an image  
2. Enters secret message  
3. Optional password encryption applied  
4. Data converted into binary  
5. Hidden using selected technique (LSB / DCT)  
6. New image generated with hidden data  

### 🔹 Decoding Process:
1. Upload encoded image  
2. Select technique  
3. Enter password (if used)  
4. Extract binary data  
5. Convert back to original message  

---

## 🔬 Techniques Used  

### 🟢 LSB (Least Significant Bit)
- Modifies pixel bits to hide data  
- High capacity  
- Minimal visual distortion  

### 🔵 DCT (Discrete Cosine Transform)
- Embeds data in frequency domain  
- More robust against compression  
- Better security
